## エージェントルール

- いかなるときも常に日本語で回答してください
  - 例
    - 考え中
    - 回答
    - ソースコード内のコメント
- "変更に 3 回失敗したら、今何をしているかを整理して、ユーザーに変更方針の確認を求めてください"
- "私の依頼に対して、具体的に何を行うつもりかをまず回答してください。"
  - その回答に対して許可が出たら実装を始めてください。

## 実装方針

- SOLID 原則、可読性、責任の分離をよく考えて実装してください
- 実装には必ずテストコードを用意してください。
  - テストコードはソースコードと同一のフォルダ内に書いてください
  - 可能な限りモックを避けて、実装側に依存関係を差し込めるように実装してください
- 複雑な実装にはコメントを残してください
- lib フォルダ内のユーティリティ関数には 十分な TSDoc を書いてください

## 開発環境

- Windows11
- Node.js v22.15.0
- npm 11.7.0
- Powershell
